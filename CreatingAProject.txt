To Create A New Java Project

1.) Open Visual Studio Code
2.) Press "CTRL + Shift + P" to open search bar
3.) in the search bar type "Create Java Project"
4.) Select No Build Tools
5.) Select a folder where you would like to keep you Project
6.) Type the name of the Project
7.) Open the Explore tab in VSCode
8.) expand the "src" file
9.) Create a new file and name it [name].Java
10.) Your ready to Code
